# CAT Preparation Strategy 

## 1. Core Strategy

You do not need multiple resources per topic. Stick to:

* 1 Concept Source
* 1 Practice Source
* 1 Mock Source

Repeat consistently.

## 7. 1-Year Study Plan

### Months 1–4 (Foundation Phase)

* Build QA fundamentals
* Start daily reading for VARC
* Begin DILR gradually
* Minimal or no mocks

---

### Months 5–8 (Strength Building)

* Start sectional tests
* Increase DILR practice
* Solve PYQs topic-wise

---

### Months 9–12 (Final Phase)

* Take 2–3 mocks per week
* Focus heavily on mock analysis
* Identify and fix weak areas

---

## 8. Key Principles

* Coaching is not necessary
* Consistency matters more than intensity
* 2–3 hours daily is sufficient if done seriously

---

## 9. Minimal Resource Stack

If following a minimal setup:

* QA: Rodha + Arun Sharma
* VARC: Gejo Speaks + daily reading
* DILR: Elites Grid
* Practice: PYQs
* Mocks: One test series

This is sufficient for top-tier preparation.
